A straight to the point Arch Linux installation guide for beginners.

Step 1: Installing Hyprland

Setting up Hyprland is quite an involved process. This section will attempt to setup a barebones system that you can build off of.

If you run into issues, I highly recommend consulting the Hyprland Wiki!

First start by installing the required Hyprland base packages:

sudo pacman -Syu hyprland sddm kitty # I recommend using Kitty as the terminal emulator as it's the default for Hyprland.

Enable sddm:

sudo systemctl enable --now sddm

If you are using an NVidia GPU you need to install the appropriate drivers:

sudo pacman -Syu nvidia # You may have to reboot after installing if you are unable to load into Hyprland.
