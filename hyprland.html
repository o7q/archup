Installing Hyprland

Setting up Hyprland is quite an involved process. This section will attempt to setup a barebones system that you can build off of. It will not hold your hand, it will rely on you to do your own research.

If you run into issues, I highly recommend consulting the Hyprland Wiki!

First start by installing the required Hyprland base packages:

sudo pacman -Syu hyprland sddm kitty # I recommend using Kitty as the terminal emulator as it's the default for Hyprland.

Enable sddm:

sudo systemctl enable --now sddm

If you are using an NVidia GPU you need to install the appropriate drivers:

sudo pacman -Syu nvidia # You may have to reboot after installing if you are unable to load into Hyprland.

Because Hyprland is modular and is entirely designed around your preference. This guide will be split up into optional sections rather than steps. It is up to you to pick and choose what you want. I would highly recommend following the Important Fixes section.

Important Fixes

Fixing audio and screenshares

To fix audio and screenshare functionality you must ensure the following services are installed and running:

sudo pacman -Syu pipewire pipewire-pulse wireplumber xdg-desktop-portal-hyprland
systemctl enable --now --user pipewire
systemctl enable --now --user pipewire-pulse
systemctl enable --now --user wireplumber

You may have to reboot your system.

Fixing authentication polkit

You can fix authentication issues by configuring hyprpolkitagent.

sudo pacman -Syu hyprpolkitagent
systemctl --user start hyprpolkitagent
exec-one = systemctl --user start hyprpolkitagent # Add this line to your Hyprland config: ~/.config/hypr/hyprland.conf

Fixing time sync

Sometimes the time won't sync correctly. To fix this enable ntp.

sudo timedatectl set-ntp true

Configuring an application launcher

Wofi will be used as the application launcher:

sudo pacman -Syu wofi

Wofi should work by default by pushing MENU + R. If it doesn't: add the following lines to your Hyprland config:

$menu = wofi --show drun
bind = $mainMod, R, exec, $menu

You can configure Wofi using its config located in: ~/.config/wofi

If you want a basic theme to get up and running, I'd recommend the Dracula Theme.
